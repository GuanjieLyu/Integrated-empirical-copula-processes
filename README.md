# Integrated Empirical Copula Test of Independence

This repository contains the **R code and reproducible materials** accompanying the paper

> **[Integrated empirical copula processes for testing independence: computation and efficiency]**

The paper proposes an integrated empirical copula-based Cramér--von Mises test for independence, develops its asymptotic theory, studies its local power and asymptotic relative efficiency, and examines its finite-sample performance through simulation and real-data analysis.

## Contents

The repository includes code for:

- constructing the integrated empirical copula process;
- computing the proposed test statistic;
- evaluating the explicit finite-sample formula derived in the paper;
- approximating critical values and p-values;
- reproducing the simulation results reported in the paper;
- reproducing the real-data application.
